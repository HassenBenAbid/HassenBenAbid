Personal Projects
Cygnus3D : Game Engine
This is an unfinished small game engine that I developed using C++/OpenGL. It has a basic rendering, some physics implementation using bullet, a loading model system with skeletal animation and also a basic lighting system.

I started this project as a mean to understand the backbones of game engines but also to challenge myself. To approach this, I started by creating a small 2D game using C++ and openGL and from there I chose the features that I want to work on and made a roadmap for myself.

C++ OpenGL GitHub
Untold Future : Rogue-Like
- This is a rogue-like top down shooter where every new restart would generate a new dungeon for the player to explore. To achieve that, I used BSP tree to create the boundaries of each room and then Delaunay triangulation to connect them. Thanks to some preset rules, the algorithm would then choose a bunch of these corridors and connect them using the A* algorithm. For the final touch we choose, from a large pool of different rooms, what kind of rooms we want and set them into their specific places.

- I also developed my own AI system using a middle ground between pluggable AI and behaviour trees to create different AI personalities. I also worked on some Coop functionalities but those weren’t implemented in the end.

All assets were imported from the Unity asset store.

C# Unity
Project Occt
