Untold Future : Rogue-Like

C# Unity
This is a rogue like top down shooter where every game is different. In it, I tried my hand with dungeon procedural generation : firstly I used BSP tree to create the boundaries of each room and the Delaunay triangulation to connect them. Then the algorithm choose, using some set of rules, a bunch of these connections and with the help of the A* algorithm it create the corridors. Finally, from a large pool of different rooms, some of them are chosen and set into the dungeon in their specific places.

I also developed my own AI system using a middle ground between pluggable AI and behaviour trees to create different AI personalities. I also worked on some Coop functionalities but those werenâ€™t implemented in the end.
