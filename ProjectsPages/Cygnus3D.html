Cygnus3D : Game Engine

C++ OpenGL
GitHub

This is a small game engine that I'm still working on using C++/OpenGL. For the moment it has a basic rendering, some physics implementation using bullet, a loading model system with skeletal animation and also a basic lighting system.The next features that will be implemented are shadow mapping and some optimisation on my code.

I started this project as a mean to understand the backbones of game engines but also to challenge myself. To approach this, I started by creating a small 2D game using C++ and openGL and from there I chose the features I want to work on and made a roadmap for myself.
