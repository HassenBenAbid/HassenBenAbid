Project Occt
Kotlin Android studio
GitHub
This is an android app compatible with any device that support miracast, that help make digital acuity test more affordable for everybody.
The app cast on any screen of your choice the different charts while transforming your smartphone into a simple remote controller. the available charts are : LogMar, Lea, tumbling but also a duochrome option and an astigmatism test. Plus a fun chart for the kids.
